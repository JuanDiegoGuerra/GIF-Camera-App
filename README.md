# 📖 Gif camera <a name="about-project"></a>

**Gif camera** is a simple,light and portable app that allows you to record a gif in a selected area of your screen.

## 💻 Getting Started <a name="getting-started"></a>

### Installation <a name="installation"></a>

- No installation needed. It's a portable app.

### Usage <a name="usage"></a>

1. After openning the app, just resize the window to the desire size you want and record your gif.

### Key Features <a name="key-features"></a>
- In this application you can do the following:
  - **Resize window**
  - **Adjust frame rate**
  - **Capture cursor**
  - **Draw Selection**
